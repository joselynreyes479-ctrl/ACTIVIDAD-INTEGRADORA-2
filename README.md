
Mini App Usuario – Joselyn Reyes

Descripción
Este proyecto es una aplicación móvil desarrollada en **Flutter**, cuyo objetivo es registrar usuarios ingresando su nombre, correo electrónico y edad, almacenando la información de forma local mediante una base de datos **SQLite**.

La aplicación presenta una interfaz moderna, amigable y colorida, pensada para facilitar el uso al usuario final.

---

 Objetivo del Proyecto
Desarrollar una aplicación móvil funcional que permita:
- Registrar datos de usuarios
- Validar formularios
- Almacenar información localmente
- Aplicar buenas prácticas de desarrollo en Flutter

---

 Tecnologías Utilizadas
- **Flutter**
- **Dart**
- **SQLite**
- **sqflite**
- **path**

---

Estructura del Proyecto
lib/
├── main.dart
└── database/
└── database_helper.dart

Funcionalidades
- Registro de usuario
- Validación de campos
- Almacenamiento local en SQLite
- Interfaz gráfica mejorada con íconos y tarjetas

---

 Ejecución del Proyecto
1. Abrir el proyecto en Visual Studio Code o Android Studio
2. Ejecutar el comando:

flutter pub get

3. Ejecutar la aplicación:

flutter run
